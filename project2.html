Wordpress Web Development

Role:

This is a low budget website, so developing an ecommerce from scratch is out of the question. Based on the clients' requirement, I choose to use a free theme template WordPress combine with WooCommerce plugin to create the ecommerce store, because both of it are free and highly customizable. By using WooCommerce, products can be create and edit through WordPress dashboard, which the client can maintain the site with little experience.
